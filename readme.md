## üöÄ Getting Started

### üß© Requirements

- Python 3.8+
- [`pystray`](https://pypi.org/project/pystray/)
- `pillow`, `pywin32`, `tkinter`
- `wkhtmltoimage.exe` in root directory for image rendering

```bash
pip install pystray pillow pywin32 pyinstaller
```
Build exe file
```bash
pyinstaller start.spec
```
